Bem-vindo!

A seguinte aplicação web trata-se de um projeto desenvolvido em Angular, utilizando Spring Boot, Hibernate e JPA com conexão banco de dados SQL Server 2019.

Com o projeto devidamente configurado em sua máquina, será possível cadastrar, pesquisar, editar e deletar clientes.

Como utilizar o programa?

Cada um dos botões abaixo representa uma das funcionalidades da CRUD, para realizar a ação necessária, basta preencher todos os campos corretamente.
